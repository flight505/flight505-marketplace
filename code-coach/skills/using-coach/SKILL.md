# Code Coach

A learning companion for Claude Code mastery.

## Available Commands

| Command | Purpose |
|---------|---------|
| `/why` | Explain the reasoning behind the last decision |
| `/teach` | Review session, teach proper programming terms |
| `/suggest` | Suggest better Claude Code workflows for current task |
| `/explain [topic]` | Explain a concept using your project as context |

Invoke commands via the Skill tool. For full skill content, invoke the corresponding skill directly.
